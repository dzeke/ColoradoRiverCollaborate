# Colorado River Collaborate Coding Efforts

This repository contains separate coding, data analysis, and visulization efforts in support of 

Each folder contains a separate, stand-on-its-own coding efforts.

Efforts share a common set of functions within the folder **AutoReadUSBRData** that auto read U.S. Bureua of Reclamation reservoir data from Reclamation's HydroData Web portal + related Bythamtry, critical elevations, and Lake Mead Water Conservation Accounting (Intentionally Created Surplus)) program data.

See additional documentation for data sources within the folder  **AutoReadUSBRData**.

## Overview

Generally the **.r**, **.Rmd, and **.pdf** files in each folder are organized as follows:

1. **.r** - Script file with the R code. Run within RStudio to conduct analysis and generate results in the figure window.
1. **.Rmd** - R markdown file with R code and documentation. Same code as the **.r** file, but when knit, generates a pdf file with embedded doccumentation, analysis, and plots.
1. **.pdf** - Pdf file generated when knit the **.Rmd** file from within RStudio.

## Directions to Reproduce Results

1. Download and install R and RStudio. See https://rstudio-education.github.io/hopr/starting.html for a tutorial.
1. Follow additional directions in the **ReadMe** file within each subfolder.

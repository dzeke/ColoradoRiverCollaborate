# Lake Mead Water Conservation Program Analysis (Intentionally Created Surplus; ICS)
## Successes, Challenges, and Potential Improvements post 2026

This folder generates four plots to support the Lake Mead Water Conservation Program Analysis. Within the Colorado River basin, the Lake Mead water conservation program is called Intentionally Created Surplus (ICS) program for Lake Mead. This program is defined in the 2007 Interim Guidelines and 2019 Drought Contingency Plans (Reclamation, 2007; 2019).

Figure 1. - Water Conservation Account balances over time.
Figure 2. - Water Conservation Account deposits and debits over time.
Figure 3. - Lake Mead storage and storage estimated if the water conservation program did not exist.
Figure 4. - Lake Mead inflow, evaporation, available water, and program deposits for which their was sufficient available water.
Note - Available Water = Inflow - Evaporation.

## View Results
Open the files:
* **LakeMeadStorageICS.pdf** - Figures 1 to 3.
* **LakeMeadInflow.pdf** - Figure 4.

## Requirements to Run
* R version 4.1.1. Download from https://cran.r-project.org/.
* R Studio 1.1.456. Download from https://www.rstudio.com/.

## Directions to Generate Results
1. Download and install R and RStudio (see requirements)
1. Within this subfolder, open the **LakeMeadWaterConservationProgramAnalysis.Rproject** file. R Studio should open.
1. Select the **LakeMeadStorageICS.Rmd** tab (R markdown file) within R Studio.
1. Just below the tab, click the **Knit** button.
1. The code will run and generate the file **LakeMeadStorageICS.pdf**. Open the pdf file to view results.
1. Select the **LakeMeadInflow.Rmd** tab (R markdown file) within R Studio.
1. Just below the tab, click the **Knit** button.
1. The code will run and generate the file **LakeMeadInflow.pdf**. Open the pdf file to view results.

## Explanation of Contents
1. **LakeMeadStorageICS.pdf** - PDF file with description of methods and Plots #1 to 3. Generated by the **LakeMeadStorageICS.Rmd** file.
1. **LakeMeadStorageICS.R** - R file with script to generate results in the R studio console. This file also has a bunch of other plots that show Lake Mead inflow using different methods.
1. **LakeMeadStorageICS.Rmd** - R mark down file with script to generate results in **LakeMeadStorageICS.pdf** file.
1. **LakeMeadInflow.pdf** - PDF file with description of methods and Plot#4. Generated by the **LakeMeadInflow.Rmd** file.
1. **LakeMeadInflow.R** - R file with script to generate results in the R studio console. This file also has a bunch of other plots that show Lake Mead inflow using different methods.
1. **LakeMeadInflow.Rmd** - R mark down file with script to generate results in **LakeMeadInflow.pdf** file.
